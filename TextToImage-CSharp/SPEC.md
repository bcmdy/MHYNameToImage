# TextToImage-CSharp - 需求文档

## 1. 项目概述

- **项目名称**：TextToImage-CSharp
- **项目类型**：命令行工具（.NET 控制台应用程序）
- **核心功能**：输入文字后生成PNG图片
- **目标用户**：需要生成文字图片的用户

## 2. 功能需求

### 2.1 核心功能

| 功能编号 | 功能描述 | 优先级 |
|---------|---------|--------|
| F1 | 读取同目录下的 TTF 字体文件 | P0 |
| F2 | 解析字体并创建 FontFace | P0 |
| F3 | 测量输入文字的宽度 | P0 |
| F4 | 生成自适应宽度的 PNG 图片 | P0 |
| F5 | 绘制文字和背景 | P0 |
| F6 | 保存图片到同目录 | P0 |
| F7 | 生成备注版图片（不同颜色） | P1 |

### 2.2 用户交互

#### 交互模式（无参数）

双击运行程序，提示输入文字，生成后3秒自动退出。

#### 命令行模式（带参数）

```
TextToImage.exe 文字      # 普通版
TextToImage.exe 文字 -m   # 普通版+备注版
```

### 2.3 图片规格

| 属性 | 普通版 | 备注版 |
|------|--------|--------|
| 文字颜色 | RGB(80, 87, 105) | RGB(100, 119, 171) |
| 背景颜色 | RGB(245, 246, 247) | RGB(245, 246, 247) |
| 图片高度 | 28px | 28px |
| 图片宽度 | 自适应 | 自适应 |
| 字体大小 | 24px | 24px |

### 2.4 部署结构

- **字体文件**：HYW.ttf 放在 exe 同目录
- **输出目录**：默认当前目录（exe 所在目录）

## 3. 技术方案

### 3.1 技术栈

- **框架**：.NET 8.0+
- **语言**：C#
- **图片处理**：SixLabors.ImageSharp + SixLabors.ImageSharp.Drawing

## 4. 验收标准

### 4.1 功能验收

- [x] 程序能正常启动，显示欢迎信息
- [x] 输入文字后能正确生成 PNG 图片
- [x] 图片文字颜色符合规格（普通版 RGB(80,87,105)，备注版 RGB(100,119,171)）
- [x] 图片背景色为 RGB(245,246,247)
- [x] 图片宽度能根据文字长度自适应
- [x] 图片高度固定为 28px
- [x] 支持 -m 参数生成备注版
- [x] 交互模式生成后3秒自动退出

### 4.2 文件验收

- [x] 生成文件名为：`文字.png` 和 `文字备注.png`
- [x] 文件格式为 PNG
- [x] 文件保存在 exe 同目录
