# TextToImage-CSharp-UI - 需求文档

## 1. 项目概述

- **项目名称**：TextToImage-CSharp-UI
- **项目类型**：Windows 桌面应用程序
- **核心功能**：提供图形界面将文字转换为 PNG 图片
- **目标用户**：需要图形化操作界面生成文字图片的用户

## 2. 功能需求

### 2.1 核心功能

| 功能编号 | 功能描述 | 优先级 |
|---------|---------|--------|
| F1 | 单文字输入生成 | P0 |
| F2 | 批量处理（支持逗号或换行分隔） | P0 |
| F3 | CSV 导入 | P0 |
| F4 | TXT 导入 | P0 |
| F5 | 生成普通版图片 | P0 |
| F6 | 生成备注版图片 | P0 |
| F7 | 自定义普通版文字颜色 | P1 |
| F8 | 自定义备注版文字颜色 | P1 |
| F9 | 自定义背景颜色 | P1 |
| F10 | 图片预览 | P1 |
| F11 | 操作日志 | P2 |
| F12 | 打开输出目录 | P2 |
| F13 | 清空输出目录 | P2 |

### 2.2 用户界面

#### 输入区域
- 单文字输入：文本框 + 生成按钮
- 批量处理：多行文本框 + 导入按钮 + 批量生成按钮

#### 选项区域
- 生成选项：普通版、备注版
- 颜色自定义：普通版文字、备注版文字、背景颜色

#### 输出区域
- 导出目录：固定为程序目录下的 output 文件夹
- 打开目录按钮
- 清空目录按钮

#### 预览区域
- 图片预览：显示生成的图片缩略图（最多 20 张）

#### 日志区域
- 操作日志：显示生成成功和失败的信息

### 2.3 图片规格

| 属性 | 普通版 | 备注版 |
|------|--------|--------|
| 文字颜色 | RGB(80, 87, 105) | RGB(100, 119, 171) |
| 背景颜色 | RGB(245, 246, 247) | RGB(245, 246, 247) |
| 图片高度 | 28px | 28px |
| 图片宽度 | 自适应 | 自适应 |
| 字体大小 | 24px | 24px |

## 3. 技术方案

### 3.1 技术栈

- **框架**：.NET 8.0
- **语言**：C#
