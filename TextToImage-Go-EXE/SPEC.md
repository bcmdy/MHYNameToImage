# NameToImage-Go-EXE - 需求文档

## 1. 项目概述

- **项目名称**：NameToImage-Go-EXE
- **项目类型**：命令行工具
- **核心功能**：输入账号名称后生成 PNG 图片
- **目标用户**：需要本地生成图片的用户

## 2. 功能需求

### 2.1 核心功能

| 功能编号 | 功能描述 | 优先级 |
|---------|---------|--------|
| F1 | 读取 TTF 字体文件 | P0 |
| F2 | 解析字体并创建 FontFace | P0 |
| F3 | 测量输入文字的宽度 | P0 |
| F4 | 生成自适应宽度的 PNG 图片 | P0 |
| F5 | 绘制文字和背景 | P0 |
| F6 | 保存图片到当前目录 | P0 |
| F7 | 生成备注版图片 | P1 |

### 2.2 命令行参数

```
-name string    账号名称
-mark           生成备注版
```

### 2.3 图片规格

| 属性 | 普通版 | 备注版 |
|------|--------|--------|
| 文字颜色 | RGB(80, 87, 105) | RGB(100, 119, 171) |
| 背景颜色 | RGB(245, 246, 247) | RGB(245, 246, 247) |
| 图片高度 | 28px | 28px |
| 图片宽度 | 自适应 | 自适应 |
| 字体大小 | 24px | 24px |

## 3. 技术方案

### 3.1 技术栈

## 4. 验收标准

### 4.1 功能验收

- [x] 程序能正常启动
- [x] 输入名称后能正确生成 PNG 图片
- [x] 图片文字颜色符合规格
- [x] 图片背景色为 RGB(245,246,247)
- [x] 图片宽度能根据文字长度自适应
- [x] 图片高度固定为 28px
