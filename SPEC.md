# 账号名称转图片生成器 - 需求文档

## 1. 项目概述

- **项目名称**：NameToImage（账号名称转图片工具）
- **项目类型**：命令行工具（.NET 控制台应用程序）
- **核心功能**：用户在 DOS 窗口输入账号名称后，自动在指定目录下生成两张 PNG 图片（普通版和备注版）
- **目标用户**：原神玩家、需要生成游戏内角色名称图片的用户

## 2. 功能需求

### 2.1 核心功能

| 功能编号 | 功能描述 | 优先级 |
|---------|---------|--------|
| F1 | 读取嵌入的 TTF 字体文件 | P0 |
| F2 | 解析字体并创建字Face | P0 |
| F3 | 测量输入文字的宽度 | P0 |
| F4 | 生成自适应宽度的 PNG 图片 | P0 |
| F5 | 绘制文字和背景 | P0 |
| F6 | 保存图片到指定目录 | P0 |
| F7 | 生成备注版图片（不同颜色） | P1 |

### 2.3 图片规格

| 属性 | 普通版 | 备注版 |
|------|--------|--------|
| 文字颜色 | RGB(80, 87, 105) | RGB(100, 119, 171) |
| 背景颜色 | RGB(245, 246, 247) | RGB(245, 246, 247) |
| 图片高度 | 28px | 28px |
| 图片宽度 | 自适应（文字宽度+边距） | 自适应 |
| 字体大小 | 24px | 24px |
| 输出格式 | PNG | PNG |

### 2.4 配置项

- **输出目录**：可通过配置文件或命令行参数指定，默认为当前目录下的 `output` 文件夹
- **字体文件**：嵌入到程序中的 HYW.ttf（华文行楷字体）

## 3. 技术方案

### 3.1 技术栈

- **框架**：.NET 8.0+
- **语言**：C#
- **图片处理**：System.Drawing.Common / ImageSharp

### 3.2 项目结构

```
NameToImage/
├── Program.cs          // 程序入口
├── ImageGenerator.cs   // 图片生成核心逻辑
├── NameToImage.csproj  // 项目文件
└── Resources/
    └── HYW.ttf        // 嵌入的字体文件
```

```csharp
public class ImageGenerator
{
    // 配置类
    public class Config
    {
        public string OutputDirectory { get; set; }

    public void GenerateImages(string name, Config config);
```

## 4. 验收标准

### 4.1 功能验收

- [ ] 程序能正常启动，显示欢迎信息
- [ ] 输入名称后能正确生成两张 PNG 图片
- [ ] 图片文字颜色符合规格（普通版 RGB(80,87,105)，备注版 RGB(100,119,171)）
- [ ] 图片背景色为 RGB(245,246,247)
- [ ] 图片宽度能根据文字长度自适应
- [ ] 图片高度固定为 28px
- [ ] 输入"退出"能正确退出程序
- [ ] 连续生成多张图片时能正常工作

### 4.2 文件验收

- [ ] 生成文件名为：`账号名.png` 和 `账号名备注.png`
- [ ] 文件格式为 PNG
- [ ] 文件保存在指定的输出目录
